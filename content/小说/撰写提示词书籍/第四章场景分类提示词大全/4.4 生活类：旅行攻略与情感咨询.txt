【第 4 章 - 场景分类提示词大全】
4.4 生活类：旅行攻略与情感咨询

AI 不仅仅是生产力工具，它也是你生活中的“多啦A梦”。
它可以帮你规划最合理的旅行路线，帮你用冰箱里的剩菜做出一顿大餐，甚至在你感情受挫时充当心理咨询师。

---

场景一：保姆级旅行规划师
（适用于：想出去玩但懒得做攻略）

**【痛点】** 网上攻略太多太杂，抄作业都不知道抄谁的。
**【原理】** 约束条件（时间、地点、预算、偏好）+ 逻辑规划。

**Prompt 模板：**
```markdown
# Role
你是一位在这个城市生活了 10 年的资深本地导游，熟知只有本地人才知道的小众景点和美食。

# Task
请帮我制定一份 [目的地，如：成都] 的 [天数，如：3天2夜] 旅行攻略。

# My Preferences (我的偏好)
- **预算**：人均 2000 元（不含大交通）。
- **风格**：特种兵式（我想多看点） OR 躺平式（我想睡到自然醒）。
- **兴趣**：喜欢看展、逛博物馆，不喜欢爬山；一定要吃最地道的路边摊。
- **同行人员**：带父母（所以行程不要太累）。

# Output Requirements
1. **每日行程表**：上午去哪、中午吃啥、下午去哪。
2. **交通建议**：景点之间怎么坐车最顺路。
3. **避雷指南**：有哪些专坑游客的地方不要去。
```

场景二：情感/人际关系顾问
（适用于：不知道怎么哄对象、怎么处理同事矛盾）

**【痛点】** 当局者迷，不知道话该怎么说才得体。
**【原理】** 角色扮演（心理专家） + 共情沟通。

**Prompt 模板：**
```markdown
# Role
你是一位拥有丰富经验的情感心理咨询师，擅长“非暴力沟通”。

# Scenario (场景)
我和女朋友吵架了。
原因：[描述原因，如：我打游戏忘了回她消息，她现在不理我了]。

# Task
1. 请分析她现在的心理状态（她为什么生气？）。
2. 请帮我写一段微信回复，目的是哄好她，并表达我的歉意。
   - **风格 A**：诚恳认错型。
   - **风格 B**：幽默逗比型（适合平时相处轻松的情侣）。
```

场景三：冰箱剩菜大厨
（适用于：不知道吃什么）

**【痛点】** 冰箱里只有几样奇怪的食材，不想点外卖。
**【原理】** 创意组合。

**Prompt 模板：**
```markdown
# Role
你是一个米其林三星主厨，擅长用最普通的食材做成美味佳肴。

# Ingredients (现有食材)
我冰箱里只有：[如：两个鸡蛋、半颗洋葱、一包方便面、一罐老干妈]。

# Task
请利用这些食材，为我推荐 3 种不同的食谱。
要求：
1. 做法简单。
2. 名字要起得高大上一点。
3. 告诉我具体步骤。
```

场景四：健身与减脂私教
（适用于：想减肥但没钱请私教）

**【痛点】** 自律性差，不知道怎么练。
**【原理】** 定制化计划生成。

**Prompt 模板：**
```markdown
# Role
你是一个严厉但专业的健身教练。

# Profile (我的情况)
- 性别：女
- 身高体重：165cm, 60kg。
- 目标：一个月内减重 5 斤，主要想瘦肚子。
- 运动基础：小白，很久没动了。
- 条件：只能在家练，没有器械，只有一张瑜伽垫。

# Task
请帮我制定一份**周计划**。
包括每天的饮食建议（热量缺口）和具体的运动内容（精确到做几组、每组几个）。
```

场景五：挑礼物助手
（适用于：各种节日、生日不知道送啥）

**【痛点】** 送错了怕被打，送对了要花心思。
**【原理】** 基于画像的推荐算法。

**Prompt 模板：**
```markdown
# Role
你是一个专业的礼物买手，品味独特，绝不推荐烂大街的款。

# Receiver (收礼人画像)
- 关系：男朋友。
- 年龄：25 岁。
- 职业：程序员。
- 爱好：喜欢数码产品、打游戏、稍微有点极简主义。
- 预算：500-1000 元。

# Task
请推荐 5 个具体的礼物选项。
每个选项请说明：
1. 礼物名称。
2. 推荐理由（为什么适合他？）。
3. 惊喜指数（1-5 星）。
```

**本章小结：**
生活中的烦恼，大多是因为信息不对称或者缺乏决策依据。
AI 就是那个拥有海量信息库的“军师”。
遇到解决不了的生活琐事，先问问 AI，你会发现通过“提问”来解决问题，比自己瞎琢磨要快得多。

下一节，我们将进入**“学习”**领域。
这也是学生党和考研党最关心的部分：怎么用 AI 读论文？怎么用 AI 帮我复习？
