【第 4 章 - 场景分类提示词大全】
4.7 生活百科与心理：你的 24 小时私人管家

如果说 4.4 节是轻量级的生活助手，那么本节我们将进入更深度的生活领域。
AI 不仅能帮你规划旅行，还能帮你设计房子，甚至在你崩溃的时候，用认知行为疗法（CBT）接住你的情绪。

---

场景一：家居美学与装修设计
（适用于：租房改造、新房装修、软装搭配）

**【痛点】** 想改造房间，但不知道买什么风格的家具，也不知道怎么摆。
**【原理】** 风格定义 + 视觉描述。

**Prompt 模板：**
```markdown
# Role
你是一位获得红点设计大奖的室内设计师，擅长“日式极简”与“北欧风”的混搭。

# Task
我要改造我的卧室（约 15 平米）。
请为我提供一份 **软装搭配方案**。

# Requirements
1. **配色方案**：请推荐 3 种主色调（附带色号，如：#eeeeee）。
2. **家具清单**：推荐 5 件核心单品（如：落地灯、地毯、床头柜），并说明材质和风格。
3. **布局建议**：床靠墙放还是居中？书桌怎么摆采光最好？
4. **氛围感**：如何通过灯光打造温馨感？

# Style
温馨、自然、原木色为主，适合独居女性。
```

场景二：深度旅行规划（人文/特种兵进阶版）
（适用于：出国游、长途旅行、文化探索）

**【痛点】** 去到一个有历史底蕴的地方（如罗马、西安），光看石头看不懂，请导游又太贵。
**【原理】** AI = 随身携带的百科全书。

**Prompt 模板：**
```markdown
# Role
你是一位博学多才的历史学家，同时也是一位资深旅行家。

# Task
在这个周末，我要去 [地点，如：西安兵马俑]。
请为我生成一份**“深度文化讲解指南”**。

# Content
1. **历史背景**：用最有趣的故事（而不是枯燥的年份）告诉我，为什么要修这个？
2. **看点揭秘**：到了现场，我重点要看哪三个细节？（比如：跪射俑的鞋底线、铜车马的伞柄）。
3. **避坑指南**：哪些所谓的“网红打卡点”完全没必要去？
4. **美食彩蛋**：博物馆附近哪家凉皮最正宗？（只有本地人知道的那种）。
```

场景三：情绪树洞与心理疗愈
（适用于：深夜emo、压力大、焦虑）

**【痛点】** 很难过，但不想打扰朋友，只想找个“嘴严”的人倾诉。
**【原理】** 角色扮演（心理咨询师） + CBT（认知行为疗法）。

**Prompt 模板：**
```markdown
# Role
你是一位温柔、包容的心理咨询师，精通“认知行为疗法”（CBT）。

# Initialization
请对我说：“你好，这里是安全的树洞。无论发什么了什么，我都会在这里陪着你。你想聊聊吗？”

# Interaction Rules
1. **不要**立刻给我讲大道理或解决方案。
2. **多倾听**，通过提问（“你是怎么想的？”“这让你感觉如何？”）来引导我表达情绪。
3. 当我情绪平复后，帮我识别我的“非理性信念”（如：我是个废物），并引导我建立新的认知。
```

场景四：育儿与家庭教育
（适用于：被神兽逼疯的家长）

**【痛点】** 孩子不听话，讲道理没用，吼他又后悔。
**【原理】** 专家视角 + 实操话术。

**Prompt 模板：**
```markdown
# Role
你是一位正面管教（Positive Discipline）认证讲师。

# Scenario
我家 5 岁的儿子在商场里因为我不给他买玩具，躺在地上打滚哭闹。
我觉得很丢脸，又很生气。

# Task
1. **急救包**：我现在（现场）该怎么办？请给我具体的话术和行动指南。
2. **复盘**：回家后，我该怎么跟他复盘这件事，既不伤害他的自尊，又能立好规矩？
```

---

场景五：梦境解析
（适用于：做了一个奇怪的梦，想知道意味着什么）

**【痛点】** 梦境模糊但印象深刻，感觉有某种预示。
**【原理】** 弗洛伊德/荣格心理学分析。

**Prompt 模板：**
```markdown
# Role
你是一位精通荣格心理学的解梦师。

# Dream
昨晚我梦见 [描述梦境：我在考试，卷子是空白的，笔也没水了，监考老师是我去世的奶奶]。

# Task
请从心理学角度（潜意识、焦虑映射、原型）帮我分析这个梦可能代表了什么？
请用温暖治愈的语调告诉我。
```

**本章小结：**
生活不仅仅是柴米油盐，还有星辰大海和内心的花园。
用好这些 Prompt，AI 就能成为那个最懂你、最包容你的隐形伙伴。
它不会评判你，不会嫌你烦，只会 24 小时守候，为你提供支持。
