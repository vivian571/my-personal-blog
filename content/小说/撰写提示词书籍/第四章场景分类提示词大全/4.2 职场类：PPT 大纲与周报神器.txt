【第 4 章 - 场景分类提示词大全】
4.2 职场类：PPT 大纲与周报神器

职场人的 80% 时间都浪费在格式调整、会议记录和回复废话邮件上。
这正是 AI 最擅长的“降维打击”领域。
用好下面这 4 个模板，你的工作效率至少提升 3 倍。

---

场景一：PPT 大纲生成器
（适用于制作演讲稿、汇报方案）

**【痛点】** 面对空白的 PPT 页面，不知道怎么排版，脑子一片空白。
**【原理】** 让 AI 充当顶级咨询顾问，不仅出内容，还出配图建议。

**Prompt 模板：**
```markdown
# Role
你是一位前麦肯锡咨询顾问，擅长制作逻辑严密、视觉美观的演示文稿。

# Task
请为主题 [在此输入你的主题，如：公司 AI 转型战略] 生成一份 10 页的 PPT 大纲。

# Constraints
- **结构**：包含封面、目录、背景、核心方案、实施计划、预期收益、结语。
- **详细度**：每一页 PPT，你需要给出以下信息：
  1. **标题**：大标题（10 字以内）。
  2. **核心观点**：这一页想要传达 Key Message（30 字以内）。
  3. **正文内容**：具体的 3-5 个 bullet points。
  4. **配图建议**：这一页应该放什么图？（如：折线图展示增长趋势、冰山图展示潜在风险）。

# Output Format
Markdown 格式，层次分明。
```

场景二：会议纪要整理神器
（适用于把乱七八糟的语音转文字变成清晰的 Action Items）

**【痛点】** 开了一小时会，录音转出的文字有 1 万字，没人想看。
**【原理】** 提取关键信息 + 过滤废话。

**Prompt 模板：**
```markdown
# Role
你是一位高效的行政秘书。

# Task
请帮我整理这份会议记录。

# Input
[在此处粘贴语音转文字的原始文本，哪怕有错别字也没关系]

# Output Requirements
请整理出以下 3 个部分：
1. **会议摘要**（100 字以内）：大家到底聊了啥？
2. **决议事项**（Decision Made）：大家达成了什么共识？
3. **待办清单**（Action Items）：谁？在什么时候？要做什么？（这是最重要的！）

# Example Output
- [ ] @张三：下周三前完成设计稿初版。
- [ ] @李四：负责联系供应商确认价格。
```

场景三：高情商邮件/回复生成器
（适用于不知道怎么回老板、怎么催客户、怎么拒绝同事）

**【痛点】** 想骂人但不敢骂，或者想催人但怕尴尬。
**【原理】** 语气风格迁移。

**Prompt 模板：**
```markdown
# Role
你是一个拥有 20 年经验的高级商务沟通专家，情商极高，擅长用最礼貌的语言表达最坚定的立场。

# Task
根据我的意图，写一封回复邮件/微信。

# My Intent (我原本想说的话)
“催一下那个乙方，怎么还不交图？都拖了三天了，再不交就扣钱！”

# Target Tone (目标语气)
- 礼貌但严肃。
- 强调时间紧迫性。
- 不卑不亢。

# Output
请生成 2 个版本供我选择：
1. **委婉版**（先礼后兵）。
2. **强硬版**（最后通牒）。
```

---

场景四：Excel 公式生成的“外挂”
（适用于记不住 VLOOKUP、IF、PivotTable 怎么用的人）

**【痛点】** 想要处理数据，但不会写复杂的公式。
**【原理】** 自然语言转代码。

**Prompt 模板：**
```markdown
# Role
你是一个 Excel 顶级专家。

# Task
请帮我写一个 Excel 公式。

# Scenario (场景描述)
我有两张表。
表 A 的 A 列是“姓名”，B 列是空白。
表 B 的 C 列是“姓名”，D 列是“工资”。
我想让表 A 的 B 列自动去匹配表 B 的 D 列（工资），如果找不到，就显示“未在职”。

# Output
1. 请直接给出公式。
2. 解释一下这个公式里的每个参数代表什么，方便我修改。
```

场景五：面试模拟器（特别版）
（适用于准备跳槽）

**【痛点】** 一个人练面试太尴尬，不知道自己哪答得不好。
**【原理】** 角色扮演 + 压力测试。

**Prompt 模板：**
```markdown
# Role
你现在是 [目标公司，如：腾讯] 的 [目标职位，如：产品经理] 面试官。你的风格是严厉、这就细节、喜欢追问。

# Task
我要进行模拟面试。请你向我提问，每次只问一个问题。
等我回答后，请先**点评**我的回答哪里不好，然后再提出下一个问题。

# Initialization
请问第一个问题：“请简单做一下自我介绍。”
```

**本章小结：**
职场 Prompt 的核心是：**把 AI 当成那个你雇不起的顶级助理**。
他可以是秘书、是 PPT 设计师、是 Excel 专家，也可以是陪练。
只要你会提要求（Role + Task），他就是免费的。

如果你不会写代码，AI 可以帮你写。如果你会写没时间，AI 可以帮你写得更快。
