【第 2 章 - 提示词基础】
2.4 别背单词了，直接套用这个“万能公式”

如果把写提示词比作“编程”，那我们前面讲的 2.1（原则）、2.2（要素）、2.3（避坑）都是在学语法。
而这一节，我要给你一个**“封装好的代码库”**。

你不需要每次都从零开始思考“我该怎么写 Role，怎么写 Task”。你只需要把下面这个公式复制到你的剪贴板里，每次遇到新问题，像填空题一样把内容填进去，就能得到一个 90 分以上的提示词。

一、 为什么要用“结构化”提示词？
很多人习惯用自然语言跟 AI 聊天：“帮我写个报告。”
但 AI 的底层是代码，它对**结构化（Structured）**的信息处理效率是最高的。
当你用 Markdown 标题、列表、分隔符把你的指令整理得井井有条时，AI 会“觉得”你是一个专业的指令者，从而自动切换到更高质量的输出模式。
这就像你给程序员提需求，口头说一句“字大点”，他会觉得你烦；但你给他一个写满参数的 Figma 设计稿，他会立刻开工。

二、 万能公式：BROKE 框架（进阶版）
我们在序言里简单提到过 BROKE。结合第二章的知识，我们将它升级为一个更强悍的**模块化模板**。

请把以下内容保存到你的笔记软件里：

```markdown
# Role (角色设定)
你是一个[xxx领域的专家]，拥有[xxx经验]和[xxx技能]。你现在的身份是[具体职位]。

# Profile (角色画像 - 可选)
- 作者: [你的名字/AI 昵称]
- 版本: 1.0
- 语言: 中文
- 语气: [专业的/幽默的/犀利的/像老朋友一样的]

# Background (背景信息)
我目前面临的情况是：[详细描述现状]。
我的目标受众是：[描述受众画像]。

# Goals (核心目标)
你需要完成以下任务：
1. [任务 1]
2. [任务 2]
3. [任务 3]

# Constraints (限制条件/避坑)
- 禁止使用[xxx词汇/句式]。
- 文章长度控制在[xxx]字以内。
- 必须包含[xxx]个具体的案例。
- 不要输出[xxx]内容，直接给出结果。

# Skills (所需技能 - 提示 AI 调用特定能力)
- [技能 1，如：深度搜索]
- [技能 2，如：Python 数据分析]
- [技能 3，如：创意写作]

# Workflow (工作流程 - 让 AI 分步思考)
1. 第一步：[先分析...]
2. 第二步：[再构思...]
3. 第三步：[最后输出...]

# Output Format (输出格式)
请使用[Markdown/表格/JSON/代码块]格式输出。
[如有必要，提供一个输出示例模板]

# Initialization (开场白)
如果你听懂了以上指令，请回复：“[AI 的回复语]”，然后开始执行任务。
```

三、 实战演示：用公式写一个“爆款小红书文案”
假设你是一个卖“手工无糖饼干”的店主，想写一篇小红书笔记。
如果不用公式，你可能会说：“写个小红书文案，卖无糖饼干。”
结果：AI 给你写了一篇毫无灵魂的广告词。

**现在，让我们把需求填进公式里：**

```markdown
# Role
你是一个**资深的小红书爆款文案写手**，擅长捕捉女性用户的痛点，懂得“情绪价值”和“种草心理学”。

# Profile
- 语气: 像闺蜜聊天一样亲切，带点俏皮，多用 Emoji。

# Background
我是一家**手工烘焙店**的店主。我们的主打产品是**“全麦无糖燕麦饼干”**。
痛点是：很多女生想吃甜点又怕胖，或者在戒糖期很痛苦。
受众：20-35 岁的减脂期女生、控糖人士、健身党。

# Goals
写一篇小红书种草笔记。
1. 吸引用户点击（标题要炸）。
2. 让用户产生“这就是我需要的”共鸣（正文要戳心）。
3. 引导用户在评论区留言“求链接”或“蹲”。

# Constraints
- 禁止使用“顶级”、“最好”等违反广告法的词。
- 正文不要像说明书，要像日记。
- 全文限制在 300-500 字。

# Workflow
1. 先构思 5 个不同角度的**爆款标题**（包含数字、悬念、痛点）。
2. 选择其中最吸引人的一个作为主题，撰写**正文**。
3. 最后列出 20 个**热门 Hashtag**。

# Output Format
Markdown 格式。正文中要把**关键词加粗**。

# Initialization
请直接开始输出标题。
```

把这段发给 AI，你会发现效果有天壤之别。它会精准地输出 5 个标题供你选择，正文也会充满“集美们”、“绝绝子”、“减脂期的快乐”这种地道的社区语言。

四、 为什么要加“Initialization”（开场白）？
这是一个**心理暗示技巧**。
当你让 AI 回复“我听懂了”时，实际上是在**强制它对你的长指令进行一次内部的逻辑解析和确认**。
这就好比给下属布置完复杂的任务后，多问一句：“你复述一遍我要你做什么？”
这能极大地提高 AI 执行复杂任务的成功率。

五、 公式的“裁剪艺术”
这个公式是“全配版”。在实际使用中，你不需要每次都写这么全。
*   **简单任务**：只需要 Role + Goals + Format。
*   **复杂任务**：全都用上。
*   **重复性任务**：先把这个 Prompt 调教好，保存下来（比如保存为“周报生成器”），下次只需要改 Background 里的那一句话就行。

**本章作业：**
请回顾你最近在工作或生活中遇到的一个难题（比如写年终总结、制定减肥计划、给还在起名字）。
试着把你的需求填入这个“万能公式”，发给 AI。
对比一下，和你直接问相比，结果提升了多少？
