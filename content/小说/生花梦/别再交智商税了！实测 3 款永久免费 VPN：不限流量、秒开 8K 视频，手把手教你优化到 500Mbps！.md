## 🛠️ 第一阶段：Cloudflare Warp 进阶优化（极速王者）

Cloudflare 出品的 Warp 被公认为业界良心。但直接使用官方客户端在部分环境下容易连不上。我们需要用到“IP优选”黑科技。

### 详细实操步骤：
1.  **下载优选工具**：访问 GitHub 或相关极客博客（如零度博客）下载 **Warp IP 优选工具**。
2.  **运行优选脚本**：
    *   解压工具包，以管理员权限运行 `warp-yip.bat`（或相应批处理文件）。
    *   在弹出窗口中选择 **“优选 IPv4 地址”**。
3.  **提取结果**：工具会自动扫描数千个节点，并生成一份 `result.csv`。
    *   **严谨提示**：我们要找的是 **Loss（丢包率）为 0%** 且 **Delay（延迟）最低** 的那个 IP。
4.  **配置连接**：
    *   下载并安装 **WireGuard** 客户端。
    *   利用 `wgcf` 工具生成个人配置文件，将配置文件中的 `Endpoint` 修改为你刚刚优选出来的最强 IP。
    *   点击连接，实测下行速率可轻松突破 **400-500Mbps**。
