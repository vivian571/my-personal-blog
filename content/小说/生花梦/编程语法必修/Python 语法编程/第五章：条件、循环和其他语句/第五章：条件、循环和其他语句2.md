# 第五章：条件、循环和其他语句

## 让程序更聪明：条件语句

咱们的程序不能只会一条道走到黑，得学会看情况办事儿。

这时候，**条件语句**就闪亮登场了！

它就像人生的十字路口，根据不同的情况，选择不同的道路。

### `if` 语句：最基本的判断

`if` 语句是最简单的条件判断。

它的意思是：“**如果**某个条件是真的，那就做这件事。”

```python
age = 18

if age >= 18:
    print("恭喜你，成年啦！可以自己做主了！")
```

看到了吗？如果 `age` 大于等于 18，程序就会打印那句话。

**注意**：`if` 后面的条件要写清楚，冒号 `:` 不能少，下面的代码块要**缩进**！

缩进是 Python 的灵魂，表示代码的从属关系，一般用 4 个空格。

### `else` 语句：如果不是这样，那就那样

光有 `if` 还不够，有时候条件不满足，咱们也得有条后路。

`else` 就是这条后路。

它的意思是：“**否则**，就做那件事。”

```python
age = 16

if age >= 18:
    print("恭喜你，成年啦！")
else:
    print("小朋友，还要等等哦！")
```

如果 `age` 不满足 `>= 18` 的条件，程序就会执行 `else` 下面的代码。

`else` 不需要条件，它就是 `if` 不满足时的备选项。

### `elif` 语句：不止两种选择

人生选择多多，程序也一样。

如果有很多种情况要判断怎么办？

用 `elif`！它是 `else if` 的缩写。

它的意思是：“**再如果**这个条件是真的，那就做这件事。”

```python
score = 85

if score >= 90:
    print("优秀！你太棒了！")
elif score >= 80:
    print("良好！继续努力！")
elif score >= 60:
    print("及格！下次加油！")
else:
    print("不及格，要好好反思一下啦！")
```

程序会从上往下依次判断 `if` 和 `elif` 的条件。

哪个条件先满足，就执行对应的代码块，然后整个 `if-elif-else` 结构就结束了。

如果所有 `if` 和 `elif` 的条件都不满足，最后才会执行 `else` 的代码（如果 `else` 存在的话）。

## 让程序动起来：循环语句

有时候，我们需要让程序重复做一些事情。

总不能手动写一百遍吧？太累了！

这时候，**循环语句**就来拯救我们了！

### `for` 循环：遍历序列的高手

`for` 循环特别擅长处理一串东西，比如列表、元组、字符串。

它的意思是：“**对于**序列里的**每一个**元素，都做这件事。”

```python
fruits = ["苹果", "香蕉", "橘子"]

for fruit in fruits:
    print("我喜欢吃：", fruit)
```

这段代码会依次把 `fruits` 列表里的每个水果拿出来，赋值给 `fruit` 变量，然后打印一句话。

`for` 循环会自动处理序列的长度，不用我们操心。

我们还可以用 `range()` 函数生成一个数字序列来循环。

```python
for i in range(5):  # range(5) 生成 0, 1, 2, 3, 4
    print("当前数字是：", i)
```

### `while` 循环：条件满足就一直做

`while` 循环和 `if` 有点像，也是看条件。

它的意思是：“**当**某个条件是真的**时候**，就一直重复做这件事。”

```python
count = 0

while count < 3:
    print("还在循环中，count =", count)
    count = count + 1  # 别忘了更新条件，否则会死循环！

print("循环结束啦！")
```

只要 `count < 3` 这个条件是真的，循环就会一直执行。

**特别注意**：一定要在 `while` 循环内部更新条件变量，否则条件永远满足，程序就会陷入**死循环**，停不下来！

## 控制循环的大师：`break` 和 `continue`

有时候，我们想在循环中途搞点特殊操作。

### `break`：立刻跳出循环

`break` 就像紧急刹车。

一旦遇到 `break`，程序会立刻**终止当前整个循环**，不管循环条件是否还满足，也不管后面还有没有代码。

```python
for i in range(10):
    if i == 5:
        print("找到 5 了，不找了！")
        break  # 立刻跳出 for 循环
    print("当前数字：", i)

print("循环外面的代码")
```

当 `i` 等于 5 时，`break` 被触发，`for` 循环直接结束。

### `continue`：跳过本次循环

`continue` 就像是跳过这一轮。

遇到 `continue`，程序会**跳过当前这次循环剩下的代码**，直接开始下一次循环。

```python
for i in range(5):
    if i == 2:
        print("遇到 2，跳过本次打印")
        continue  # 跳过本次循环的 print
    print("当前数字：", i)
```

当 `i` 等于 2 时，`continue` 被触发，后面的 `print` 语句被跳过，直接进入 `i` 等于 3 的循环。

## `pass` 语句：啥也不干，占个位置

有时候，语法上需要写点东西，但我们暂时还没想好写啥，或者就是不需要做任何事。

这时候，可以用 `pass`。

`pass` 是一个空语句，它**不做任何事情**，只是为了保持代码结构的完整性。

```python
def my_function():
    pass  # 函数体不能为空，先用 pass 占个位

if some_condition:
    pass  # 这里暂时不需要处理
else:
    print("处理其他情况")
```

`pass` 就像一个占位符，告诉 Python：“这里我知道要写代码，但现在先空着。”

## 总结一下

这一章我们学了 Python 里非常重要的控制流语句：

*   **条件语句 (`if`, `elif`, `else`)**：让程序根据不同情况做出不同反应。
*   **循环语句 (`for`, `while`)**：让程序重复执行任务。
*   **循环控制 (`break`, `continue`)**：更灵活地控制循环流程。
*   **占位语句 (`pass`)**：保持代码结构完整。

掌握了这些，你的程序就能变得更加智能和强大啦！继续加油！
