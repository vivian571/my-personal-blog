## 🚀 谷歌白皮书揭秘的五大提示词技巧

### 技巧一：零样本提示（Zero-shot Prompting）

**<font color='purple'>什么是零样本提示？</font>**

就是直接告诉AI你想要什么，不给任何例子。

**<font color='blue'>适合场景：</font>**

简单明确的任务。

**<font color='green'>实战示例：</font>**

```
将以下文本翻译成英文：
中国有着悠久的历史和灿烂的文化。
```

**<font color='red'>效果一般！</font>**

因为没有给AI明确的指导和例子。

### 技巧二：少样本提示（Few-shot Prompting）

**<font color='purple'>什么是少样本提示？</font>**

给AI几个例子，让它学会你想要的模式。

**<font color='blue'>适合场景：</font>**

需要特定格式或风格的输出。

**<font color='green'>实战示例：</font>**

```
将以下句子翻译成幽默风格的英文：

例子1：
中文：我今天很累。
英文：I'm so tired today that my bed is sending me love letters.

例子2：
中文：这个菜很好吃。
英文：This dish is so delicious that my taste buds are doing a happy dance.

现在翻译：
中文：中国有着悠久的历史和灿烂的文化。
```

**<font color='red'>效果提升！</font>**

因为AI通过例子学会了你想要的幽默风格。

### 技巧三：思维链提示（Chain of Thought, CoT）

**<font color='purple'>什么是思维链提示？</font>**

引导AI像人类一样一步步思考问题。

**<font color='blue'>适合场景：</font>**

复杂推理、数学问题、逻辑分析。

**<font color='green'>实战示例：</font>**

```
问题：小明有5个苹果，小红有3个苹果，小刚有2个苹果。如果小明给了小红2个苹果，小红又给了小刚1个苹果，现在每个人有多少个苹果？

请一步步思考：
1. 首先，列出初始状态
2. 然后，计算第一次转移后的状态
3. 最后，计算第二次转移后的状态
```

**<font color='red'>效果显著！</font>**

因为AI被引导进行了清晰的步骤推理，减少了错误。

### 技巧四：思维树提示（Tree of Thoughts, ToT）

**<font color='purple'>什么是思维树提示？</font>**

让AI同时探索多条思路，评估每条路径的可行性。

**<font color='blue'>适合场景：</font>**

需要创造性思维、多方案比较的复杂问题。

**<font color='green'>实战示例：</font>**

```
问题：如何提高一家咖啡店的销售额？

请使用思维树方法，探索三个不同方向的解决方案：
1. 从产品角度思考
   1.1 [具体想法]
   1.2 [评估这个想法的优缺点]
   1.3 [可行性分析]

2. 从营销角度思考
   2.1 [具体想法]
   2.2 [评估这个想法的优缺点]
   2.3 [可行性分析]

3. 从客户体验角度思考
   3.1 [具体想法]
   3.2 [评估这个想法的优缺点]
   3.3 [可行性分析]

最后，综合评估三个方向，推荐最佳解决方案。
```

**<font color='red'>效果惊人！</font>**

因为AI被引导从多个角度思考问题，并进行了深入分析。

### 技巧五：ReAct提示（Reasoning + Acting）

**<font color='purple'>什么是ReAct提示？</font>**

结合推理和行动，让AI先思考再行动。

**<font color='blue'>适合场景：</font>**

需要多步骤操作、信息搜索和决策的任务。

**<font color='green'>实战示例：</font>**

```
任务：帮我规划一次三天的北京旅行。

请按照以下步骤思考和行动：
1. 思考：我需要考虑哪些因素来规划这次旅行？
2. 行动：列出北京必去的景点和美食。
3. 思考：如何安排这些景点才最合理？
4. 行动：制定三天的详细行程安排。
5. 思考：有哪些可能的问题和应对方案？
6. 行动：提供旅行建议和注意事项。
```

**<font color='red'>效果最佳！</font>**

因为AI被引导进行了深度思考和具体行动的结合。

# 提示词工程师工具箱
import openai
import json

class PromptEngineer:
    def __init__(self, api_key):
        self.api_key = api_key
        openai.api_key = api_key
        
    def zero_shot_prompt(self, task, content):
        """零样本提示"""
        prompt = f"{task}\n\n{content}"
        return self._get_completion(prompt)
    
    def few_shot_prompt(self, task, examples, content):
        """少样本提示"""
        examples_text = "\n\n".join([f"例子{i+1}:\n{ex['input']}\n{ex['output']}" 
                              for i, ex in enumerate(examples)])
        prompt = f"{task}\n\n{examples_text}\n\n现在处理:\n{content}"
        return self._get_completion(prompt)
    
    def chain_of_thought(self, problem, steps):
        """思维链提示"""
        steps_text = "\n".join([f"{i+1}. {step}" for i, step in enumerate(steps)])
        prompt = f"问题: {problem}\n\n请一步步思考:\n{steps_text}"
        return self._get_completion(prompt)
    
    def tree_of_thoughts(self, problem, branches):
        """思维树提示"""
        branches_text = ""
        for i, branch in enumerate(branches):
            branches_text += f"{i+1}. 从{branch['perspective']}角度思考\n"
            for j, sub_step in enumerate(branch['sub_steps']):
                branches_text += f"   {i+1}.{j+1} {sub_step}\n"
        
        prompt = f"问题: {problem}\n\n请使用思维树方法，探索不同方向的解决方案:\n{branches_text}\n最后，综合评估所有方向，推荐最佳解决方案。"
        return self._get_completion(prompt)
    
    def react_prompt(self, task, steps):
        """ReAct提示"""
        steps_text = ""
        for i, step in enumerate(steps):
            step_type = "思考" if i % 2 == 0 else "行动"
            steps_text += f"{i+1}. {step_type}: {step}\n"
        
        prompt = f"任务: {task}\n\n请按照以下步骤思考和行动:\n{steps_text}"
        return self._get_completion(prompt)
    
    def _get_completion(self, prompt):
        """调用API获取回复"""
        try:
            response = openai.ChatCompletion.create(
                model="gpt-3.5-turbo",
                messages=[
                    {"role": "system", "content": "你是一个专业的AI助手。"},
                    {"role": "user", "content": prompt}
                ],
                temperature=0.7
            )
            return response.choices[0].message.content
        except Exception as e:
            return f"错误: {str(e)}"

# 使用示例

    print("零样本提示结果:\n", result1)
    
    # 少样本提示示例
    examples = [
        {"input": "中文: 我今天很累。",

        {"input": "中文: 这个菜很好吃。",

    ]
    result2 = engineer.few_shot_prompt(
        "将以下句子翻译成幽默风格的英文:", 
        examples,
        "中文: 中国有着悠久的历史和灿烂的文化。"
    )
    print("\n少样本提示结果:\n", result2)
    
    # 思维链提示示例
    steps = [

        steps

    print("\n思维链提示结果:\n", result3)
    
    # 思维树提示示例
    branches = [
        {
            "perspective": "产品",
            "sub_steps": ["具体想法", "评估这个想法的优缺点", "可行性分析"]
        },
        {
            "perspective": "营销",

# 提示词质量评估工具
def evaluate_prompt(prompt):
    score = 0
    checks = {
        "具体明确": lambda p: 5 if len(p.split()) > 15 else 0,
        "步骤引导": lambda p: 5 if any(str(i) in p for i in range(1, 6)) else 0,
        "示例提供": lambda p: 5 if "例子" in p or "示例" in p or "例如" in p else 0,
        "格式要求": lambda p: 5 if any(format_word in p for format_word in ["格式", "风格", "结构"]) else 0,
        "目标明确": lambda p: 5 if any(goal_word in p for goal_word in ["目标", "目的", "要求"]) else 0
    }
    
    results = {}
    for check_name, check_func in checks.items():
        check_score = check_func(prompt)
        score += check_score
        results[check_name] = "✅" if check_score > 0 else "❌"
    
    quality = "优秀" if score >= 20 else "良好" if score >= 15 else "一般" if score >= 10 else "需改进"
    
    print(f"提示词质量评估结果：{quality} (得分：{score}/25)")
    for check_name, result in results.items():
        print(f"{check_name}: {result}")
    
    return score, quality

# 测试你的提示词
prompt = input("请输入你的提示词：")
evaluate_prompt(prompt)
