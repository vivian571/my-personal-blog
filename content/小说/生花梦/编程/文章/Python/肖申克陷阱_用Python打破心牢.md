## 用Python记录你的“越狱”之路

**<span style={{ color: 'blue', fontWeight: 'bold' }}>光说不练假把式！</span>**

为了帮助大家更好地实践“小失控”，我写了个简单的Python脚本。

它可以帮你记录和追踪你挑战舒适区的行动。

**<span style={{ color: 'green', fontWeight: 'bold' }}>代码来了，接好！</span>**

```python
import datetime
import json
import os

# 定义数据存储文件
DATA_FILE = 'comfort_zone_challenges.json'

def load_challenges():
    """加载已有的挑战记录"""
    if os.path.exists(DATA_FILE):
        try:
            with open(DATA_FILE, 'r', encoding='utf-8') as f:
                return json.load(f)
        except json.JSONDecodeError:
            print(f"警告：{DATA_FILE} 文件格式错误或为空，将创建新的记录。")
            return []
    return []

def save_challenges(challenges):
    """保存挑战记录到文件"""
    with open(DATA_FILE, 'w', encoding='utf-8') as f:
        json.dump(challenges, f, ensure_ascii=False, indent=4)

def add_challenge(challenges):
    """添加一个新的挑战记录"""
    print("\n--- 添加新的舒适区挑战 --- ")
    challenge_desc = input("请输入你完成的挑战描述：")
    
    while True:
        try:
            difficulty = int(input("给这次挑战打个分 (1-5, 5为最难)："))
            if 1 <= difficulty <= 5:
                break
            else:
                print("评分必须在1到5之间！")
        except ValueError:
            print("请输入有效的数字！")
            
    feelings = input("完成后的感受如何？(例如：紧张但兴奋、很有成就感等)：")
    learnings = input("从中学到了什么？(例如：原来我可以做到、认识了新朋友等)：")
    
    challenge_record = {
        'date': datetime.datetime.now().strftime('%Y-%m-%d %H:%M:%S'),
        'description': challenge_desc,
        'difficulty': difficulty,
        'feelings': feelings,
        'learnings': learnings
    }
    
    challenges.append(challenge_record)
    save_challenges(challenges)
    print("\n**<span style={{ color: 'green', fontWeight: 'bold' }}>挑战记录成功！你又向自由迈出了一步！</span>**")

def view_challenges(challenges):
    """查看所有挑战记录"""
    if not challenges:
        print("\n**<span style={{ color: 'orange', fontWeight: 'bold' }}>你还没有记录任何挑战呢，快去行动吧！</span>**")
        return
        
    print("\n--- 我的舒适区挑战记录 --- ")
    for i, record in enumerate(challenges, 1):
        print(f"\n记录 {i}:")
        print(f"  日期: {record['date']}")
        print(f"  挑战: {record['description']}")
        print(f"  难度: {'★' * record['difficulty']}{'☆' * (5 - record['difficulty'])}")
        print(f"  感受: {record['feelings']}")
        print(f"  收获: {record['learnings']}")
    print("------------------------")

def main():
    challenges = load_challenges()
    
    while True:
        print("\n--- 舒适区突破追踪器 --- ")
        print("1. 添加新的挑战记录")
        print("2. 查看所有挑战记录")
        print("3. 退出")
        choice = input("请选择操作 (输入数字)：")
        
        if choice == '1':
            add_challenge(challenges)
        elif choice == '2':
            view_challenges(challenges)
        elif choice == '3':
            print("\n**<span style={{ color: 'purple', fontWeight: 'bold' }}>持续行动，打破心牢！下次再见！</span>**")
            break
        else:
            print("\n**<span style={{ color: 'red', fontWeight: 'bold' }}>无效输入，请输入 1, 2 或 3。</span>**")

if __name__ == "__main__":
    main()

```

**<span style={{ color: 'orange', fontWeight: 'bold' }}>这个脚本能帮你干啥？</span>**

1.  **<span style={{ color: 'purple', fontWeight: 'bold' }}>记录你的“勇敢时刻”：</span>** 每次做了点突破舒适区的事，就记下来。
2.  **<span style={{ color: 'red', fontWeight: 'bold' }}>量化你的感受和难度：</span>** 给挑战打分，写下感受和收获。
3.  **<span style={{ color: 'blue', fontWeight: 'bold' }}>回顾你的成长足迹：</span>** 随时查看记录，看看自己走了多远。
4.  **<span style={{ color: 'green', fontWeight: 'bold' }}>数据化你的“越狱”进度：</span>** 把抽象的“突破”变成看得见的数据。

**<span style={{ color: 'orange', fontWeight: 'bold' }}>怎么用？</span>**

*   把代码保存成一个 `.py` 文件（比如 `challenge_tracker.py`）。
*   在你的电脑上安装 Python 环境（如果还没有的话）。
*   打开命令行（终端），进入到你保存文件的目录。
*   运行命令：`python challenge_tracker.py`
*   跟着提示操作就行了！
