## 如何安装CursorPowerTools？

![安装步骤](https://cdn.jsdelivr.net/gh/microsoft/fluentui-emoji/assets/Laptop/3D/laptop_3d.png)

安装过程非常简单，只需几步就能完成：

### 步骤1：下载Cursor编辑器

如果你还没有安装Cursor，先从官方网站下载并安装最新版本。

CursorPowerTools支持Windows、macOS和Linux版本的Cursor。

### 步骤2：安装CursorPowerTools

打开终端，执行以下命令：

```bash
# 克隆仓库
git clone https://github.com/CursorPowerTools/cursor-power-tools.git

# 进入目录
cd cursor-power-tools

# 安装依赖
npm install

# 构建插件
npm run build
```

### 步骤3：配置Cursor

打开Cursor编辑器，按下`Ctrl+Shift+P`（macOS上是`Cmd+Shift+P`）打开命令面板。

输入"extensions"，选择"Install Extension from Location"。

选择刚才构建好的插件目录，点击确认。

重启Cursor，你就会发现所有Pro功能都已解锁！

### 连接本地模型

要使用本地模型，你需要先安装Ollama或LM Studio等工具。

然后在CursorPowerTools设置中，配置模型API地址和参数：

```json
{
  "cursorPowerTools.localModel": {
    "enabled": true,
    "apiUrl": "http://localhost:11434/api",
    "modelName": "codellama:13b",
    "temperature": 0.2
  }
}
```

配置完成后，所有AI功能都会使用你的本地模型，速度更快，安全性更高。

### 团队共享设置

如果你在团队中使用CursorPowerTools，可以创建共享配置文件：

```json

```

这样团队成员可以使用统一的设置，提高协作效率。
