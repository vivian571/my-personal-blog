# 保姆级教程！零代码小白用 Trae 做出 3 个爆款工具实录

## 前言

你是否曾经有过这样的想法：如果我会编程该多好，我就能做出自己想要的工具和应用。但是学习编程需要大量时间和精力，对于零基础的小白来说门槛太高了。

好消息是，字节跳动推出的 Trae IDE 让这一切变得简单！Trae 是一款 AI 原生集成开发环境，它集成了强大的 AI 模型（如 Claude 3.5 和 GPT-4o），支持中文交流，完全免费，而且最重要的是 —— 零代码小白也能用！

今天，我将手把手教你如何用 Trae 打造 3 个实用又炫酷的工具，全程无需写一行代码，只需要清晰描述你的需求即可！

## 准备工作：安装 Trae

在开始制作工具前，我们需要先安装 Trae：

1. 访问官网 https://www.trae.com.cn/ 并点击"下载"按钮
2. 下载并安装到你的电脑上
3. 首次启动时，可以选择从 VS Code 或 Cursor 导入配置（如果你之前用过这些工具）
4. 注册并登录账号（推荐，这样可以保存你的项目）

## 认识 Trae 界面

Trae 的界面主要分为四个区域：

1. **左侧边栏**：包含文件浏览器、搜索、Git 仓库、网页预览、调试等功能
2. **文件区**：显示项目中的所有文件和文件夹
3. **代码编辑区**：中间最大的区域，用于查看和编辑代码
4. **AI 交互区**：右侧的对话区域，有两种模式：
   - **Builder 模式**：告诉 AI 你想要什么应用，它会从零开始构建
   - **Chat 模式**：可以向 AI 提问、寻求建议

对于零代码小白，我们将主要使用 **Builder 模式**，这是我们的魔法工具！

## 工具一：个人简历生成器

### 步骤 1：创建新项目

1. 打开 Trae，点击"新建项目"
2. 在右侧 Builder 模式中，输入以下描述：

```
我想创建一个简历生成器网站，用户可以输入个人信息（如姓名、联系方式、教育背景、工作经历、技能等），然后生成一份美观的简历，可以导出为 PDF。界面要简洁美观，有多种模板可选。
```

3. Trae 会自动开始创建项目，生成所需的所有代码文件

### 步骤 2：查看和运行项目

1. 当 Trae 完成代码生成后，在右侧会看到运行指令
2. 点击 Trae 提供的命令按钮，或在终端中输入：

```
npm install
npm start
```

3. 点击出现的"Open WebView"按钮，预览你的简历生成器

### 步骤 3：自定义和改进

如果你想修改某些功能，只需在 Builder 模式中描述你的需求：

```
请将简历模板增加到 5 种，并添加一个暗色模式切换按钮
```

Trae 会自动修改代码并实现你的需求！

## 工具二：智能待办事项管理器

### 步骤 1：创建新项目

1. 返回 Trae 首页，创建新项目
2. 在 Builder 模式中输入：

```
我需要一个智能待办事项管理器，具有以下功能：
1. 添加、编辑、删除任务
2. 设置任务优先级和截止日期
3. 根据优先级和截止日期自动排序任务
4. 任务完成后可以标记为已完成
5. 数据保存在本地存储中，下次打开仍然存在
6. 界面要美观，支持拖拽排序
```

### 步骤 2：运行和测试

1. 等待 Trae 生成代码
2. 按照提示运行项目：

```
npm install
npm start
```

3. 在 WebView 中测试你的待办事项管理器

### 步骤 3：增强功能

想要增加更多功能？只需在 Builder 模式中继续描述：

```
请添加以下功能：
1. 任务分类功能（工作、个人、学习等）
2. 添加提醒功能，可以设置提醒时间
3. 增加数据导出功能，可以导出为CSV文件
```

## 工具三：AI 图片生成器网站

### 步骤 1：创建项目

1. 创建新项目
2. 在 Builder 模式中输入：

```
我想创建一个 AI 图片生成网站，用户可以输入文字描述，然后调用 Stable Diffusion API 生成对应的图片。需要有以下功能：
1. 文字输入框，用于描述想要生成的图片
2. 图片尺寸和风格选择
3. 生成按钮
4. 生成历史记录
5. 图片下载功能
6. 美观的界面设计
```

### 步骤 2：配置 API

Trae 会提示你需要配置 API 密钥：

1. 按照 Trae 的指导，注册一个 Stable Diffusion API 账号
2. 获取 API 密钥
3. 在项目中配置 API 密钥（Trae 会指导你如何安全地存储它）

### 步骤 3：运行和优化

1. 运行项目
2. 测试图片生成功能
3. 根据需要进一步优化：

```
请优化用户界面，添加以下功能：
1. 图片生成过程中显示加载动画
2. 增加图片分享到社交媒体的功能
3. 添加图片编辑功能，如简单的裁剪、滤镜等
```

## 常见问题解答

**问：我完全不懂编程，能用好 Trae 吗？**
答：完全可以！Trae 的设计初衷就是让非程序员也能创造软件。你只需要清晰描述你的需求，剩下的交给 Trae。

**问：Trae 生成的工具安全吗？**
答：非常安全。Trae 生成的工具默认在本地运行，你的数据不会被上传到云端。

**问：如果 Trae 生成的代码有问题，我该怎么办？**
答：只需在 Chat 模式下描述问题，如"代码运行时出现错误，提示XXX"，Trae 会帮你诊断并修复问题。

**问：我可以将 Trae 生成的工具分享给别人使用吗？**
答：当然可以！你可以将项目部署到网络上，或者打包成应用分享给他人。Trae 还可以指导你如何进行部署。
