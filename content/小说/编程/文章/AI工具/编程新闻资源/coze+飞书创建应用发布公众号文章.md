## 🛠️ 第一步：配置Coze机器人，变身内容生成大师

首先，我们需要在Coze平台创建一个专门用于生成公众号内容的AI机器人。

### 1. 注册并登录Coze

访问 [https://www.coze.cn](https://www.coze.cn)（国内版）或 [https://www.coze.com](https://www.coze.com)（国际版），使用手机号或邮箱注册登录。

### 2. 创建专属公众号助手

点击右上角的「**+创建Bot**」按钮，填写基本信息：

- 名称：「公众号内容助手」（或你喜欢的任何名字）
- 头像：选择一个专业的形象
- 描述：「专门用于生成和发布公众号文章的AI助手」

### 3. 设置Bot的专业能力

在「**提示词**」部分，我们需要给Bot设定明确的角色和能力。复制下面的提示词：

```
你是一名专业的公众号内容创作助手，擅长以下工作：
1. 根据用户提供的主题，生成高质量的公众号文章
2. 文章风格活泼有趣，语言接地气
3. 自动添加适当的emoji表情和分段
4. 生成适合公众号的标题和导语
5. 提供文章的SEO优化建议

当用户请求你生成文章时，请按照以下格式输出：
【标题】：吸引人的标题
【导语】：简短的导语，吸引读者继续阅读
【正文】：分段清晰的正文内容
【结语】：有号召力的结束语
【标签】：3-5个相关标签
```

**<font color='purple'>这个提示词是核心，决定了你的Bot能生成什么样的内容！</font>**

### 4. 添加必要的工具

在「**工具**」部分，添加以下工具增强Bot的能力：

- **网页浏览器**：让Bot能获取最新资讯
- **图片生成**：为文章创建配图
- **文件读取**：可以读取你上传的参考资料

### 5. 创建发布工作流

这是最关键的一步！我们要创建一个工作流，实现内容从生成到发布的自动化：

1. 点击左侧的「**工作流**」选项
2. 点击「**创建工作流**」，命名为「公众号发布流程」
3. 在画布上添加以下节点：
   - **开始节点**（默认已有）
   - **内容生成节点**（使用AI生成内容）
   - **内容审核节点**（检查内容质量）
   - **飞书连接节点**（将在下一步配置）

4. 连接这些节点，形成一个完整的工作流

```javascript
// 工作流节点连接示例代码
const workflow = {
  name: "公众号发布流程",
```

## 🚀 第二步：配置飞书，搭建发布通道

有了内容生成机器人，接下来我们需要配置飞书，建立一个从Coze到公众号的发布通道。

### 1. 注册飞书开发者账号

访问 [飞书开放平台](https://open.feishu.cn/)，注册并登录开发者账号。

### 2. 创建自定义应用

1. 在飞书开发者平台，点击「**创建应用**」
2. 填写应用信息：
   - 名称：「公众号发布助手」
   - 描述：「连接Coze和公众号的自动化发布工具」
   - 选择「**企业自建应用**」类型

### 3. 获取应用凭证

创建应用后，在应用详情页面，找到并记录以下信息：

- **App ID**
- **App Secret**

这两个凭证在后续连接时会用到。

### 4. 配置应用权限

在「**权限管理**」页面，添加以下权限：

- **消息与群组**：用于发送通知
- **云文档**：用于存储文章内容
- **网络钩子**：用于接收Coze的请求

### 5. 创建飞书自动化流程

1. 进入飞书「**工作台**」
2. 点击「**自动化**」应用
3. 创建新的自动化流程，命名为「公众号文章发布」
4. 设置触发条件为「**收到网络请求**」
5. 添加以下操作步骤：
   - **接收内容**：从Coze接收文章内容
   - **格式转换**：将内容转换为公众号格式
   - **调用公众号API**：发布到指定公众号

```python
# 飞书自动化流程示例代码
import requests
import json

def handle_webhook(request):
    # 接收来自Coze的内容
    content = request.json.get('content')
    title = request.json.get('title')
    
    # 格式转换
    formatted_content = format_for_wechat(content)
    
    # 调用公众号API发布
    publish_to_wechat(title, formatted_content)
    
    return {"status": "success"}

def format_for_wechat(content):
    # 将内容转换为公众号支持的格式
    # 这里是简化的示例
    return content.replace('\n', '<br>')

def publish_to_wechat(title, content):
    # 调用公众号API
    # 需要替换为实际的公众号API调用
    api_url = "https://api.weixin.qq.com/cgi-bin/..."
    token = "YOUR_ACCESS_TOKEN"
    
    data = {
        "title": title,
        "content": content,
        "thumb_media_id": "MEDIA_ID",
        "digest": "文章摘要",
        "show_cover_pic": 1
    }
    
    response = requests.post(f"{api_url}?access_token={token}", json=data)
    return response.json()
```

**<font color='red'>注意：实际的公众号API调用需要你先完成公众号开发者认证，并获取相应的access_token！</font>**

## 🔄 第三步：连接Coze和飞书，打通全流程

现在，我们需要将Coze和飞书连接起来，实现端到端的自动化。

### 1. 在Coze中配置飞书连接

回到Coze的工作流编辑页面，选择「**飞书连接节点**」，进行以下配置：

1. 选择「**HTTP请求**」类型
2. 请求方式选择「**POST**」
3. URL填写飞书自动化流程的webhook地址
4. 请求头添加：
   ```
   Content-Type: application/json
   ```
5. 请求体模板：
   ```json
   {
     "title": "{{title}}",
     "content": "{{content}}",
     "author": "Coze机器人",
     "publish_time": "{{current_time}}"
   }
   ```

### 2. 测试连接

1. 在Coze工作流页面，点击「**测试**」按钮
2. 输入测试内容："帮我生成一篇关于AI应用的公众号文章"
3. 观察工作流执行情况，确保每个节点都正常工作
4. 检查飞书是否收到请求，以及是否成功转发到公众号

### 3. 设置定时任务

如果你想定期发布内容，可以在飞书自动化中添加定时触发器：

1. 编辑之前创建的自动化流程
2. 添加新的触发条件：「**定时触发**」
3. 设置触发时间，例如每周一早上9点
4. 保存设置

**<font color='blue'>这样，即使你在睡觉，系统也能按时为你发布优质内容！</font>**

## 🎁 实用技巧与注意事项

### 技巧一：内容多样化

不要让你的Bot只生成一种风格的内容。可以在提示词中设置多种模板，让内容更丰富多样。

### 技巧三：备份机制

设置内容自动备份到飞书文档或其他云存储，防止意外丢失。
