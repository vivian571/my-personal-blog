# 第一章：Python 3 基础入门

嘿，朋友们！欢迎来到 Python 的奇妙世界！准备好开启一段轻松愉快的编程旅程了吗？别担心，Python 这家伙超级友好，就算你是编程小白也能轻松上手。咱们这就出发！

## 初识 Python

### Python 简介

想象一下，你想让电脑帮你干点活，比如算个数、处理个文件，甚至搞个网站。这时候，你就需要一种能跟电脑沟通的语言，而 **Python** 就是这样一种 **超级流行**、**简单易学** 的编程语言！

它就像是编程界的“瑞士军刀”，功能强大，啥都能干！而且它的语法特别像咱们平时说话，读起来贼顺畅，写起来也舒服。

### Python 历史

话说在 1989 年的圣诞节，一位名叫 **Guido van Rossum** 的荷兰大佬觉得当时的编程语言有点不给力，要么太复杂，要么太难用。于是乎，他决定自己动手，丰衣足食，搞一个 **简单**、**优雅**、**功能强大** 的新语言。

为了纪念他喜欢的喜剧团体“Monty Python”，他就给这个新语言取名叫 Python。是不是挺随意的？哈哈！经过这么多年的发展，Python 已经从一个“个人项目”成长为全球最受欢迎的编程语言之一啦！

### Python 优缺点

没有什么是完美的，Python 也不例外。咱们来盘点一下它的优缺点：

**优点（为啥大家都爱它）：**

*   **简单易学**：语法像英语，读起来不费劲，写起来也快。
*   **免费开源**：随便用，随便改，不用花一分钱！社区还特别活跃，有啥问题一堆人帮你。
*   **功能强大**：库（别人写好的代码模块）特别多，想干啥几乎都有现成的工具，不用自己从头造轮子。
*   **跨平台**：Windows、Mac、Linux 都能跑，兼容性杠杠的。
*   **面向对象**：这是一种编程思想，能让你的代码更有条理，更容易维护。
*   **可扩展性强**：可以把其他语言（比如 C/C++）写的代码“粘”进来用。

**缺点（偶尔也让人头疼）：**

*   **运行速度相对较慢**：跟 C/C++ 这种编译型语言比起来，Python 解释执行的速度会慢一些。不过别担心，对于大部分应用场景来说，这点速度差异根本不是事儿！而且有很多优化手段。
*   **代码不能加密**：因为是解释型语言，源代码容易被看到。当然，也有一些方法可以混淆代码。
*   **全局解释器锁（GIL）**：这玩意儿让 Python 在同一时间只能真正利用一个 CPU 核心来执行线程，对多线程并行计算不太友好。不过，可以通过多进程或者其他库来绕开这个问题。

## 数据类型

电脑处理的都是数据，但数据也分很多种。Python 里常见的数据类型有这么几种：

### 整型 (int)

就是 **整数**，不带小数点的，比如 `1`, `100`, `-5`, `0`。

```python
my_age = 18
count = -99
zero = 0
```

### 浮点型 (float)

就是 **带小数点的数**，比如 `3.14`, `-0.5`, `2.718`。

```python
pi = 3.14159
price = 99.9
negative_float = -0.01
```

**注意**：有时候计算浮点数可能会有微小的误差，这是计算机二进制表示的限制，大部分情况不用太在意。

### 复数 (complex)

这个可能数学课上见过，就是形如 `a + bj` 的数，`a` 是实部，`b` 是虚部，`j` 是虚数单位。编程里用的相对少一些，了解一下就行。

```python
complex_num = 3 + 4j
print(complex_num.real) # 输出实部 3.0
print(complex_num.imag) # 输出虚部 4.0
```

### 数据类型转换

有时候我们需要把一种类型的数据变成另一种类型，比如把用户输入的文字（字符串）变成数字（整型或浮点型）来计算。

*   `int()`：把别的类型转成 **整型**。
*   `float()`：把别的类型转成 **浮点型**。
*   `str()`：把别的类型转成 **字符串**。

```python
num_str = "100"
num_int = int(num_str) # 把字符串 "100" 转成整数 100

float_num = 3.14
int_num = int(float_num) # 把浮点数 3.14 转成整数 3 (小数部分直接丢掉)

number = 666
str_num = str(number) # 把整数 666 转成字符串 "666"
```

**注意**：不是啥都能转的，比如你不能把 `"hello"` 转成整数，会报错！

## 常量

常量就是 **不变的量**。比如圆周率 π ≈ 3.14159...，它就是个常量。

在 Python 里，没有像其他语言那样严格意义上的常量定义方式。但是，程序员们有个 **约定俗成** 的规矩：**用全大写的变量名来表示常量**，比如 `PI = 3.14159`。

虽然你还是可以偷偷改掉 `PI` 的值，但大家看到全大写的变量，就知道：“哦，这哥们想让它当常量使，咱就别动它了。” 这是一种 **代码规范**。

```python
PI = 3.14159
MAX_CONNECTIONS = 10
```

## 变量与关键字

### 变量

变量就像一个 **带标签的盒子**，你可以往里面放东西（数据），并且可以随时更换里面的东西。

在 Python 里，给变量赋值非常简单，用 `=` 就行了。

```python
message = "你好，世界！" # 把字符串 "你好，世界！" 放进名叫 message 的盒子里
count = 10           # 把整数 10 放进名叫 count 的盒子里
price = 19.9         # 把浮点数 19.9 放进名叫 price 的盒子里

# 变量里的东西可以变
count = count + 1    # 现在 count 盒子里的东西变成 11 了
message = "Hello, Python!" # message 盒子里的东西也换了
```

Python 是 **动态类型** 语言，意思是你不必事先告诉 Python 这个变量要装什么类型的数据，它会自动识别。而且，同一个变量，先装整数，后装字符串，也没问题！

```python
x = 10      # x 现在是整数
print(x)
x = "哈哈"  # x 现在变成字符串了
print(x)
```

### 变量名称

给变量起名字也是有讲究的，不能瞎起。

**规则：**

1.  只能包含 **字母**、**数字** 和 **下划线 `_`**。
2.  **不能以数字开头**。
3.  **区分大小写** (`myVar` 和 `myvar` 是两个不同的变量)。
4.  **不能使用 Python 的关键字** (后面会讲)。

**建议（让代码更好看懂）：**

*   名字要有意义，**见名知意**。比如用 `user_name` 而不是 `un`。
*   推荐使用 **下划线命名法** (snake_case)，比如 `first_name`, `item_count`。
*   避免使用容易混淆的字母，比如小写 `l` 和大写 `I`，数字 `0` 和大写 `O`。

**合法的变量名：** `name`, `age`, `user_id`, `_internal_var`, `MAX_VALUE`
**非法的变量名：** `1st_place` (数字开头), `user-name` (包含连字符), `class` (关键字)

### 关键字 (Keywords)

关键字是 Python 语言 **内部保留** 的一些单词，它们有特殊的含义和功能，**你不能用它们来当变量名**。

Python 的关键字不多，可以通过下面的代码查看：

```python
import keyword
print(keyword.kwlist)
```

常见的关键字有：`if`, `else`, `for`, `while`, `def`, `class`, `import`, `return`, `True`, `False`, `None` 等等。眼熟一下就好，后面学到具体功能时自然就记住了。

## 语句 (Statements)

语句就是 **让 Python 执行一个操作的指令**。一行代码通常就是一个语句。

比如：

```python
print("Hello")      # 这是一个打印语句
x = 5             # 这是一个赋值语句
if x > 0:         # 这是一个条件语句 (包含子语句)
    print("Positive") # 这是条件语句块里的打印语句
```

Python 代码是 **从上到下** 一行一行执行的（除非遇到像 `if`, `for`, `while` 这样的控制流语句）。

## 表达式 (Expressions)

表达式就是 **能计算出一个值的代码片段**。它可以是：

*   一个 **字面量** (直接写出来的值)：`10`, `3.14`, `"Hello"`
*   一个 **变量**：`my_age`, `price`
*   **运算符和操作对象的组合**：`5 + 3`, `count * 2`, `age >= 18`
*   一个 **函数调用**：`int("100")`, `len("abc")`

简单来说，**能丢到 `print()` 里打印出结果的，基本都是表达式**。

```python
result = 5 + 3       # 5 + 3 是一个表达式，计算结果是 8
is_adult = age >= 18 # age >= 18 是一个表达式，计算结果是 True 或 False
message = "Hello" + " " + "World" # "Hello" + " " + "World" 是一个表达式
```

## 运算符和操作对象

### 什么是运算符和操作对象

想象一下数学里的 `5 + 3`。

*   `+` 就是 **运算符 (Operator)**，它表示要做什么操作（这里是加法）。
*   `5` 和 `3` 就是 **操作对象 (Operand)**，它们是参与运算的东西。

Python 里的运算符有很多种，用来执行不同的计算和操作。

### 算术运算符

就是咱们小学数学学的加减乘除那些。

*   `+`：加法 (`1 + 2` 结果是 `3`)
*   `-`：减法 (`5 - 3` 结果是 `2`)
*   `*`：乘法 (`4 * 5` 结果是 `20`)
*   `/`：除法 (`10 / 2` 结果是 `5.0`，**注意结果总是浮点数**)
*   `//`：整除 (`10 // 3` 结果是 `3`，只保留整数部分)
*   `%`：取余 (模运算) (`10 % 3` 结果是 `1`，就是除法的余数)
*   `**`：幂运算 (`2 ** 3` 结果是 `8`，就是 2 的 3 次方)

```python
a = 10
b = 3
print(a + b)  # 13
print(a - b)  # 7
print(a * b)  # 30
print(a / b)  # 3.333...
print(a // b) # 3
print(a % b)  # 1
print(a ** b) # 1000
```

### 比较运算符

用来比较两个值的大小或者是否相等，结果总是 **布尔值 (Boolean)**：`True` (真) 或 `False` (假)。

*   `==`：等于 (`5 == 5` 结果是 `True`, `5 == 3` 结果是 `False`)
*   `!=`：不等于 (`5 != 3` 结果是 `True`, `5 != 5` 结果是 `False`)
*   `>`：大于 (`5 > 3` 结果是 `True`)
*   `<`：小于 (`5 < 3` 结果是 `False`)
*   `>=`：大于等于 (`5 >= 5` 结果是 `True`)
*   `<=`：小于等于 (`5 <= 3` 结果是 `False`)

```python
x = 10
y = 5
print(x == y)  # False
print(x != y)  # True
print(x > y)   # True
print(x <= y)  # False
```

### 赋值运算符

用来给变量赋值。

*   `=`：最基本的赋值 (`x = 5`)
*   `+=`：加法赋值 (`x += 2` 等价于 `x = x + 2`)
*   `-=`：减法赋值 (`x -= 2` 等价于 `x = x - 2`)
*   `*=`：乘法赋值 (`x *= 2` 等价于 `x = x * 2`)
*   `/=`：除法赋值 (`x /= 2` 等价于 `x = x / 2`)
*   `//=`：整除赋值 (`x //= 2` 等价于 `x = x // 2`)
*   `%=`：取余赋值 (`x %= 2` 等价于 `x = x % 2`)
*   `**=`：幂赋值 (`x **= 2` 等价于 `x = x ** 2`)

这些 `+=`, `-=` 之类的运算符叫 **增强赋值运算符**，能让代码更简洁一点。

```python
count = 10
count += 5 # count 现在是 15
print(count)

price = 100
price *= 0.8 # 打八折，price 现在是 80.0
print(price)
```

### 位运算符

这个稍微进阶一点，是 **直接操作数字在内存中的二进制位**。一般在底层开发、优化性能或者特定算法中会用到，初学阶段可以先 **跳过** 或者 **简单了解**。

*   `&`：按位与
*   `|`：按位或
*   `^`：按位异或
*   `~`：按位取反
*   `<<`：左移位
*   `>>`：右移位

如果你对二进制不熟悉，现在不用管它，以后需要了再学也不迟！

### 逻辑运算符

用来连接多个比较表达式，判断最终的真假。

*   `and`：**与**，两边都为 `True`，结果才为 `True`。
*   `or`：**或**，两边只要有一个为 `True`，结果就为 `True`。
*   `not`：**非**，把 `True` 变 `False`，把 `False` 变 `True`。

```python
age = 25
has_ticket = True

# 年龄大于 18 并且有票才能进
can_enter = age > 18 and has_ticket
print(can_enter) # True

score = 80
is_vip = False

# 分数大于 60 或者 是 VIP 就能通过
can_pass = score > 60 or is_vip
print(can_pass) # True

is_raining = False
print(not is_raining) # True (因为 not False 就是 True)
```

### 成员运算符

用来判断一个东西 **是否在** 另一个东西（通常是字符串、列表、元组等序列）里面。

*   `in`：如果在里面，结果为 `True`。
*   `not in`：如果不在里面，结果为 `True`。

```python
my_string = "hello world"
print('h' in my_string)      # True
print('z' in my_string)      # False
print('world' in my_string)  # True

my_list = [1, 2, 3, 4, 5]
print(3 in my_list)          # True
print(6 not in my_list)      # True
```

### 身份运算符

用来比较两个变量 **是不是指向内存中同一个对象**。注意，它比较的是 **身份 (identity)**，而不是 **值 (value)**。

*   `is`：如果指向同一个对象，结果为 `True`。
*   `is not`：如果指向不同的对象，结果为 `True`。

```python
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a == b)    # True (值相等)
print(a is b)    # False (a 和 b 是两个不同的列表对象，虽然内容一样)

print(a == c)    # True (值相等)
print(a is c)    # True (a 和 c 指向的是同一个列表对象)

print(a is not b) # True
```

**什么时候用 `is`？**

通常用来判断一个变量是否是 `None` (Python里的空值)。

```python
x = None
if x is None:
    print("x 是空的")
```

对于数字和短字符串，Python 为了优化，可能会让值相同的变量指向同一个对象，但不要依赖这个特性，**判断值是否相等，老老实实用 `==`**。

## 运算符优先级

当一个表达式里有多个运算符时，谁先算谁后算？这就涉及到 **优先级**。

跟咱们数学里先乘除后加减类似，Python 也有自己的规则：

1.  `**` (幂运算) 优先级最高。
2.  `~`, `+`, `-` (一元运算符，比如取反 `~x`，正号 `+x`，负号 `-x`)
3.  `*`, `/`, `//`, `%` (乘、除、整除、取余)
4.  `+`, `-` (加、减)
5.  `<<`, `>>` (位移)
6.  `&` (按位与)
7.  `^` (按位异或)
8.  `|` (按位或)
9.  `==`, `!=`, `>`, `>=`, `<`, `<=` (比较运算符)
10. `is`, `is not` (身份运算符)
11. `in`, `not in` (成员运算符)
12. `not` (逻辑非)
13. `and` (逻辑与)
14. `or` (逻辑或)

**记不住怎么办？**

没关系！**用括号 `()`**！括号拥有 **最高的优先级**，可以明确指定运算顺序，还能让代码更容易看懂。

```python
result = 5 + 3 * 2 # 先算 3 * 2 = 6，再算 5 + 6 = 11
print(result)

result = (5 + 3) * 2 # 先算括号里的 5 + 3 = 8，再算 8 * 2 = 16
print(result)

# 就算你记得优先级，加个括号也更清晰
can_access = (age >= 18 and has_permission) or is_admin
```

**强烈建议：** 对于复杂的表达式，**多用括号**，准没错！

## 字符串操作

字符串就是 **一串文本**，用单引号 `' '` 或者双引号 `" "` 括起来。

```python
name = 'Alice'
message = "Hello, Python!"
long_string = '''这是一个
可以跨行的
长字符串。''' # 三引号可以写多行字符串
```

字符串有很多常用的操作：

*   **拼接**：用 `+` 把字符串连起来。
    ```python
    greeting = "Hello" + " " + "World"
    print(greeting) # 输出 "Hello World"
    ```
*   **重复**：用 `*` 重复字符串。
    ```python
    line = '-' * 20
    print(line) # 输出 --------------------
    ```
*   **获取长度**：用 `len()` 函数。
    ```python
    s = "abcde"
    print(len(s)) # 输出 5
    ```
*   **索引 (Indexing)**：获取单个字符，从 `0` 开始计数。
    ```python
    s = "Python"
    print(s[0]) # 输出 'P'
    print(s[1]) # 输出 'y'
    print(s[-1]) # 输出 'n' (负数表示从末尾开始数)
    ```
*   **切片 (Slicing)**：获取一部分子字符串 `[start:end:step]` (包头不包尾)。
    ```python
    s = "Python"
    print(s[0:2]) # 输出 'Py' (从索引 0 到 2，不包括 2)
    print(s[2:])  # 输出 'thon' (从索引 2 到末尾)
    print(s[:4])  # 输出 'Pyth' (从开头到索引 4，不包括 4)
    print(s[::-1]) # 输出 'nohtyP' (步长为 -1，实现字符串反转)
    ```
*   **常用方法**：字符串有很多内置的方法，比如转换大小写、查找、替换等。
    ```python
    text = " hello world "
    print(text.upper())      # 输出 " HELLO WORLD " (转大写)
    print(text.lower())      # 输出 " hello world " (转小写)
    print(text.strip())      # 输出 "hello world" (去除两端空格)
    print(text.find('o'))    # 输出 5 (查找 'o' 第一次出现的位置)
    print(text.replace('l', 'X')) # 输出 " heXXo worXd " (替换)
    print(text.split(' '))   # 输出 ['', 'hello', 'world', ''] (按空格分割成列表)
    ```

字符串操作非常常用，多练练就熟了！

## 注释 (Comments)

注释就是 **写给程序员看的文字**，Python 解释器会 **忽略** 它们，不会执行。

注释的作用：

*   **解释代码**：说明这段代码是干嘛的，为什么这么写。
*   **暂时禁用代码**：不想执行某段代码，又不想删掉，可以先注释掉。
*   **留下标记**：比如 `TODO:` 提醒自己这里还有活没干完。

Python 里有两种注释：

1.  **单行注释**：以 `#` 开头，从 `#` 到行尾的内容都是注释。
    ```python
    # 这是单行注释，解释下面的代码
    x = 10 # 这也是单行注释，可以在代码后面加
    ```
2.  **多行注释**：严格来说 Python 没有专门的多行注释语法，但可以用 **三个单引号 `''' '''`** 或 **三个双引号 `""" """`** 包裹起来的字符串来充当。这种字符串如果没有被赋值给变量，解释器也会忽略它。
    ```python
    '''
    这是一个多行注释，
    可以写很多行的说明文字。
    '''
    def my_function():
        """这也是一个多行注释，通常用在函数或类的开头，
        叫做文档字符串 (docstring)，用来解释函数/类的作用。
        """
        pass # pass 是一个空语句，啥也不干
    ```

**写注释是个好习惯！** 虽然 Python 代码已经很易读了，但对于复杂的逻辑或者关键的部分，加上注释能让别人（以及未来的你）更容易理解你的代码。
