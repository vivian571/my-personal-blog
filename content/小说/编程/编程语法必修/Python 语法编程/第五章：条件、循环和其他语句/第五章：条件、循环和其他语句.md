## import 的使用

在 Python 里，有大量现成的功能被打包成了 **<font color='purple'>模块 (Module)</font>**，想用就得先 **<font color='red'>导入 (import)</font>**。

### import 语句

最基本的导入方式就是 `import` 关键字：

```python
# 导入整个模块
import math

# 使用模块中的函数
result = math.sqrt(16)  # 计算平方根
print(result)  # 输出 4.0
```

**<font color='orange'>记住：</font>** 导入后，必须用 `模块名.函数名` 的方式来调用！

### 使用逗号导入

想一次性导入多个模块？用逗号分隔就行：

```python
# 一次导入多个模块
import math, random, time

print(math.pi)  # 圆周率
print(random.randint(1, 10))  # 1到10之间的随机整数
print(time.time())  # 当前时间戳
```

**<font color='blue'>一行搞定多个导入，代码更简洁！</font>**

还有 `from...import` 语法，可以直接导入模块中的特定函数或变量：

```python
# 只导入需要的函数
from math import sqrt, pi

# 直接使用函数名，不需要加模块前缀
print(sqrt(25))  # 输出 5.0
print(pi)  # 输出 3.141592653589793
```

**<font color='green'>按需导入，用起来更方便！</font>**

## 别样的赋值

Python 的赋值操作可不只是简单的 `变量 = 值`，还有一些骚操作等着你来解锁！

### 序列解包

想一次性给多个变量赋值？序列解包来帮忙：

```python
# 一次性给多个变量赋值
a, b, c = 1, 2, 3
print(a, b, c)  # 输出 1 2 3

# 交换两个变量的值，不需要临时变量
x, y = 10, 20
x, y = y, x  # 交换 x 和 y 的值
print(x, y)  # 输出 20 10

# 从列表或元组解包
coordinates = (3, 4)
x, y = coordinates
print(f"坐标：({x}, {y})")  # 输出 坐标：(3, 4)
```

**<font color='red'>这操作，简直是优雅到飞起！</font>**

### 链式赋值

想给多个变量赋相同的值？链式赋值一步到位：

```python
# 链式赋值，一次给多个变量赋相同的值
x = y = z = 100
print(x, y, z)  # 输出 100 100 100
```

**<font color='purple'>一行代码搞定，就是这么简单！</font>**

### 增量赋值

想让变量在原来的基础上增加或减少？增量赋值运算符来帮忙：

```python
# 增量赋值
score = 0
score += 10  # 等同于 score = score + 10
print(score)  # 输出 10

score -= 3   # 等同于 score = score - 3
print(score)  # 输出 7

score *= 2   # 等同于 score = score * 2
print(score)  # 输出 14

score /= 2   # 等同于 score = score / 2
print(score)  # 输出 7.0
```

**<font color='blue'>简洁又直观，代码量少一半！</font>**

## 语句块

Python 用 **<font color='red'>缩进</font>** 来表示代码块，这点和其他语言用花括号 `{}` 不太一样。

**<font color='orange'>缩进有多重要？</font>** 在 Python 里，缩进不是装饰，而是语法的一部分！缩进错了，代码就会报错或者逻辑错乱。

```python
# 正确的缩进示例
if True:
    print("这是第一层缩进")
    if True:
        print("这是第二层缩进")
    print("回到第一层缩进")
print("没有缩进，回到最外层")
```

**<font color='green'>记住：</font>** 同一个代码块的缩进必须一致，通常是 4 个空格（按 Tab 键自动转换）。

## 条件语句

生活中，我们经常需要根据不同情况做出不同决策。比如：
- 如果下雨，就带伞
- 如果饿了，就吃饭
- 如果困了，就睡觉

在 Python 中，这种决策靠的就是 **<font color='red'>条件语句</font>**！

### 布尔变量的作用

条件语句的核心是 **<font color='blue'>布尔值 (Boolean)</font>**，也就是 `True` 或 `False`。

```python
is_raining = True
is_hungry = False

print(is_raining)  # 输出 True
print(is_hungry)   # 输出 False
print(type(is_raining))  # 输出 <class 'bool'>
```

**<font color='purple'>布尔值就像开关，决定代码走哪条路！</font>**

### if 语句

`if` 语句是最基本的条件语句，语法是：`if 条件: 执行的代码`

```python
# 基本的 if 语句
age = 20

if age >= 18:
    print("你已经成年了！")
    print("可以看成人电影了！")

print("这行代码无论如何都会执行")
```

**<font color='orange'>注意：</font>** `if` 后面的条件为 `True` 时，缩进的代码块才会执行。

### else 子句

想在条件不满足时执行另一段代码？`else` 子句来帮忙：

```python
# if-else 语句
age = 16

if age >= 18:
    print("你已经成年了！")
    print("可以看成人电影了！")
else:
    print("你还未成年！")
    print("只能看儿童电影哦！")

print("这行代码无论如何都会执行")
```

**<font color='green'>if 和 else，两条路总有一条会走！</font>**

### elif 子句

如果有多个条件需要判断呢？用 `elif`（else if 的缩写）：

```python
# if-elif-else 语句
score = 85

if score >= 90:
    print("优秀！")
elif score >= 80:
    print("良好！")
elif score >= 60:
    print("及格！")
else:
    print("不及格，继续努力！")
```

**<font color='red'>多条件判断，elif 来搞定！</font>**

### 嵌套代码块

条件语句还能嵌套使用，形成更复杂的判断逻辑：

```python
# 嵌套条件语句
age = 20
has_id = True

if age >= 18:
    print("年龄符合要求")
    if has_id:
        print("身份证已验证，可以进入")
    else:
        print("请出示身份证")
else:
    print("未成年不得入内")
```

**<font color='blue'>条件套条件，逻辑更精细！</font>**

## 更多操作

### 同一性运算符

Python 中有两种判断相等的方式：
- `==`：判断值是否相等
- `is`：判断是否是同一个对象（内存地址相同）

```python
# == 和 is 的区别
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a == b)  # 输出 True（值相等）
print(a is b)  # 输出 False（不是同一个对象）
print(a is c)  # 输出 True（是同一个对象）
```

**<font color='purple'>== 看值，is 看身份证号！</font>**

### 比较字符串和序列

字符串、列表等序列也可以直接比较：

```python
# 字符串比较
print("apple" < "banana")  # 输出 True（按字母顺序比较）

# 列表比较
print([1, 2] < [1, 3])  # 输出 True（逐元素比较）
print([1, 2, 3] < [1, 2])  # 输出 False（长度不同时，先比较共有部分）
```

**<font color='green'>序列比较，按顺序一个一个来！</font>**

### 布尔运算符

多个条件可以用布尔运算符组合：
- `and`：两边都为 True，结果才为 True
- `or`：两边只要有一个为 True，结果就为 True
- `not`：取反，True 变 False，False 变 True

```python
# 布尔运算符
age = 25
income = 8000

# and 运算符（两个条件都满足）
if age > 18 and income > 5000:
    print("你可以申请信用卡！")

# or 运算符（满足任意一个条件）
if age > 60 or income > 10000:
    print("你可以享受特殊优惠！")

# not 运算符（取反）
is_weekend = False
if not is_weekend:
    print("工作日，要上班！")
```

**<font color='red'>布尔运算符，让条件判断更灵活！</font>**

### 断言

`assert` 语句用来检查一个条件，如果条件为 False，程序会抛出异常并停止运行：

```python
# assert 语句
age = 10
assert age > 0, "年龄必须是正数"  # 条件为 True，继续执行
# assert age > 18, "必须年满18岁"  # 条件为 False，会抛出 AssertionError
```

**<font color='orange'>断言就像门卫，不符合条件就拦住！</font>**

## 循环

有时候，我们需要重复执行某些操作，比如打印 1 到 100 的数字。难道要写 100 行 `print` 语句吗？

当然不用！这时候就需要 **<font color='red'>循环</font>** 出场了！

### while 循环

`while` 循环会在条件为 True 时重复执行代码块：

```python
# while 循环
count = 1

while count <= 5:
    print(f"当前计数：{count}")
    count += 1  # 别忘了更新条件，否则会无限循环！

print("循环结束")
```

**<font color='blue'>while 循环，条件为真就一直转！</font>**

### for 循环

`for` 循环用来遍历序列（如列表、元组、字符串等）中的元素：

```python
# for 循环遍历列表
fruits = ["苹果", "香蕉", "橘子"]

for fruit in fruits:
    print(f"我喜欢吃{fruit}")

# for 循环遍历字符串
for char in "Python":
    print(char)

# 使用 range() 生成数字序列
for i in range(1, 6):  # 1, 2, 3, 4, 5
    print(i)
```

**<font color='green'>for 循环，挨个遍历不放过！</font>**

### 循环遍历字典元素

字典的循环遍历有多种方式：

```python
# 遍历字典
student = {"name": "小明", "age": 18, "major": "计算机科学"}

# 遍历键
for key in student:
    print(key)

# 遍历值
for value in student.values():
    print(value)

# 同时遍历键和值
for key, value in student.items():
    print(f"{key}: {value}")
```

**<font color='purple'>字典遍历，键值随你选！</font>**

### 迭代工具

Python 提供了一些强大的迭代工具，让循环更加灵活。

#### iter() 和 next()

`iter()` 函数可以创建一个迭代器，`next()` 函数用于获取迭代器的下一个元素：

```python
# 使用 iter() 和 next()
numbers = [1, 2, 3, 4, 5]
iter_obj = iter(numbers)

print(next(iter_obj))  # 输出 1
print(next(iter_obj))  # 输出 2
print(next(iter_obj))  # 输出 3
# 当没有更多元素时，next() 会抛出 StopIteration 异常
```

**<font color='red'>手动控制迭代，想取就取！</font>**

#### enumerate()

`enumerate()` 函数可以在遍历序列的同时获取元素的索引：

```python
# 使用 enumerate()
fruits = ["苹果", "香蕉", "橘子"]

for index, fruit in enumerate(fruits):
    print(f"索引 {index}: {fruit}")

# 指定起始索引
for index, fruit in enumerate(fruits, start=1):
    print(f"第 {index} 个水果是: {fruit}")
```

**<font color='blue'>索引和值一起拿，enumerate 来帮忙！</font>**

#### zip()

`zip()` 函数可以同时遍历多个序列：

```python
# 使用 zip()
names = ["小明", "小红", "小刚"]
ages = [18, 17, 19]
scores = [90, 85, 92]

for name, age, score in zip(names, ages, scores):
    print(f"{name}今年{age}岁，考了{score}分")
```

**<font color='green'>多序列齐步走，zip 来组合！</font>**

### 跳出循环

有时候，我们需要在特定条件下提前结束循环或跳过当前迭代。

#### break 语句

`break` 语句可以立即跳出整个循环：

```python
# 使用 break 提前结束循环
for i in range(1, 11):
    if i == 5:
        print("遇到 5 了，跳出循环！")
        break
    print(i)

print("循环结束")
```

**<font color='orange'>break 一出，循环结束！</font>**

#### continue 语句

`continue` 语句可以跳过当前迭代，直接进入下一次迭代：

```python
# 使用 continue 跳过当前迭代
for i in range(1, 11):
    if i % 2 == 0:  # 如果是偶数
        continue  # 跳过，不执行下面的代码
    print(i)  # 只打印奇数
```

**<font color='purple'>continue 一出，跳过不走！</font>**

### 循环中的 else 子句

Python 的循环还有一个特殊功能：`else` 子句。当循环正常结束（不是通过 `break` 跳出）时，会执行 `else` 块：

```python
# 循环的 else 子句
for i in range(1, 6):
    print(i)
else:
    print("循环正常完成！")

# 使用 break 的情况
for i in range(1, 6):
    if i == 3:
        break
    print(i)
else:
    print("这句话不会执行，因为循环被 break 中断了")
```

**<font color='red'>循环正常结束才执行 else，被 break 就不执行！</font>**

## pass 语句

有时候，我们需要在语法上写一个代码块，但暂时不想实现具体功能，这时就可以用 `pass` 语句作为占位符：

```python
# 使用 pass 作为占位符
def todo_function():
    pass  # 先占个位，以后再实现

if age < 18:
    pass  # 暂时不处理未成年的情况
else:
    print("成年人")
```

**<font color='blue'>pass 就是空操作，啥也不干，就占个位！</font>**
