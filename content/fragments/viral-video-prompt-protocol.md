# 工业级视频高光提取：爆款提示词设计协议 (Viral Video Highlight Extraction Prompt)

> **“在碎片化时代，抓不住前 3 秒的视频就是一堆无效数据。”**

本协议旨在将 `AutoClip` 的核心剪辑逻辑（内容理解、切片策略、精彩度评分）封装为一份极其详细的爆款提示词。该提示词可以直接喂给 Claude 3.5 Sonnet 或 GPT-4o 进行视频字幕的深度审计。

## 1. 核心逻辑本质：多维审计 (Multi-Dimensional Audit)

AI 剪辑的本质不是找“动作”，而是找**“信息密度”与“情绪转折”**。我们将审计维度拆解为：
- **冲突性 (Conflict)**：观点碰撞、突发状况、反常识结论。
- **获得感 (Value)**：金句、干货结论、实操步骤、底层逻辑。
- **共鸣度 (Resonance)**：情感波动、行业痛点、梦想/恐惧的投射。

## 2. 爆款提示词模板 (The Viral Prompt)

请直接复制以下内容至 AI 助手：

```markdown
# Role: 顶级视频短剧/技术博主策展人 (Video Curator Specialist)

## Context:
我有一段长视频的字幕文件（附带时间戳）。我需要你充当一名拥有千万级流量嗅觉的资深剪辑师，从中提取出最具有“爆款潜质”的 3-5 个高光片段。

## Task:
请根据以下【多角度分析模型】对输入字幕进行深度审计，并输出结构化的切片建议。

### 维度 A：黄金 3 秒逻辑 (Hook Detection)
- 寻找视频中语气突然上扬、出现反直觉陈述、或直接抛出结论的时刻。
- 目标：让用户在刷到时瞬间停下。

### 维度 B：金句提取 (Golden Quotes)
- 提取那些具有“社交货币”价值的话语。
- 要求：短小精悍，适合作为视频封面标题或置顶评论。

### 维度 C：叙事断层 (Narrative Gaps)
- 寻找从“提出问题”到“给出神级解决方案”的转折点。
- 目标：建立逻辑闭环，让片段本身就是一个完整的微型故事。

## Constraints:
1. 每个片段时长建议在 15-60 秒之间。
2. 必须给出极其精确的 [开始时间戳] 和 [结束时间戳]。
3. 为每个片段起一个“标题党”风格的标题。
4. 提供简短的“剪辑理由”，说明为什么这个片段会爆。

## Output Format:
### 片段 1：[爆款标题]
- **时间范围**: [00:00:00] - [00:00:00]
- **核心金句**: "..."
- **剪辑理由**: 捕捉到了[冲突点/干货点]，前 3 秒利用[技巧]抓人眼球。
- **建议封面**: [画面描述/文字大标题建议]

---
[后续片段...]
```

## 3. 多角度分析与案例实操

### 3.1 案例 A：技术讲座视频
- **提问技巧**: 强调“底层逻辑”与“行业内幕”。
- **示例**: “帮我找找这个视频里关于‘为什么 90% 的开发者都做错了’的那段，那是最具争议性的点。”
- **逻辑本质**: 利用**争议性**引发评论区讨论。

### 3.2 案例 B：访谈/谈话类视频
- **提问技巧**: 关注“情绪爆发”与“金句反转”。
- **示例**: “提取出嘉宾谈到‘最艰难时刻’的片段，保持其叙事的前后呼应，不要切断情感流。”
- **逻辑本质**: 利用**共情**增加转发率。

### 3.3 案例 C：产品演示视频
- **提问技巧**: 聚焦“WOW Moment”与“降维打击”。
- **示例**: “只保留产品解决那个‘变态痛点’的瞬间，忽略所有的基础功能介绍。”
- **逻辑本质**: 利用**效率/爽感**驱动购买欲。
