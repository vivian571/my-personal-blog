以安，原来你打算在 Windows 下运行它！这是一个非常关键的细节，因为 **Windows 系统有一个特别的“防坑指南”**。

在项目的官方说明里，作者特意标出了一个红色的警告 ⚠️：**Windows 用户千万别用刚才那个最简单的安装法**，因为 OpenClaw 目前在 Windows 下会遇到一个兼容性小 Bug（叫做 `spawn npm ENOENT`，也就是系统找不到默认的安装工具）。

所以，在 Windows 下，我们不能买“成品”，必须换上**“手工拼装模式”**（官方叫“方式二：从源码安装”）。别怕，对于六年级小朋友来说，这就像是买乐高回来自己拼，一共只需要四步指令：

### 🎒 Windows 专属保姆级教程

**🛠️ 准备工作小检查：**
在开始前，你的 Windows 电脑里需要有两把“螺丝刀”：一个是 `git`（用来下载源码），另一个是 `pnpm`（用来组装代码）。*(如果没装，我们得先装这两个，稍后可以告诉我)*。

#### 第一步：把“零件箱”搬回家

打开 Windows 的命令行窗口（按键盘上的 `Win` 键，搜 `cmd` 或 `PowerShell`，然后回车打开黑框框），把下面这行复制进去敲回车：

```cmd
git clone https://github.com/BytePioneer-AI/openclaw-china.git

```

*💡 意思就是：去 GitHub 仓库把整个中国版插件包下载到你的电脑里。*

#### 第二步：走进这个文件夹

```cmd
cd openclaw-china

```

#### 第三步：开始组装零件

在这个文件夹里，依次输入这两句魔法咒语（输入第一句回车，等它跑完再输入第二句）：

```cmd
pnpm install
pnpm build

```

*💡 这一步可能需要一两分钟，屏幕上会跳出很多进度条，系统正在帮你把零散的代码变成能运行的工具。*

#### 第四步：挂载工具并召唤“傻瓜向导”

最后，告诉你的 OpenClaw 新工具装在哪里，并启动设置向导：

```cmd
openclaw plugins install -l ./packages/channels
openclaw china setup

```

*💡 敲完最后一句回车，你就会看到我们刚才说的**傻瓜选择题界面**了！依然是用键盘上下键选择“钉钉”，然后按提示去拿密码填入。*
